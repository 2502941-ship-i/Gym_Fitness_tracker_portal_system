FitTrack is a web-based fitness management system designed to help users track their health, workouts, and communication with trainers. It provides features like BMI calculation, nutrition tracking, workout suggestions, and real-time chat between clients and trainers.

---

## 🎯 Modules Implemented

### 🔹 Major Module
- Dashboard
- BMI Calculator
- Workout Tracking

### 🔹 Minor Module
- Trainer Chat System 💬
- Nutrition Tracking

---

## 🛠️ Technologies Used

- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Node.js (Express)
- **Database:** MySQL
- **Design Tool:** Figma
